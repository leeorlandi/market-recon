How it works
01 —

Type a topic

Plain language, a capability area, or a pasted PRD. Market Recon normalizes the input, checks scope, and asks two questions: your role and the decision you're trying to make.

02 —

Agent team deploys

Dimension analysts research each capability lane in parallel. A scout finds what they miss. A buyer analyst reads G2, Reddit, and forums. A fact-checker resolves citations before anything gets written.

03 —

Gets smarter every run

A world model records every player and angle. The next run inherits it — the scout always searches past what's known. After Run 4, a longitudinal brief synthesizes market movement across all runs.

Requirements
  • ✓ Claude Code v2.1.32 or later — check with claude --version
  • ✓ Agent teams enabled — add CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1 to your ~/.claude/settings.json
  • ✓ Web search access — Market Recon searches the live web on every run
Install
terminal project-scoped install
# Clone the repo and open it in Claude Code
git clone https://github.com/leeorlandi/market-recon
claude market-recon/

# Then run your first recon
/market-recon AI SDRs

To make /market-recon available in every project, run ./install.sh from the repo root.
